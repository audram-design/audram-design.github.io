Website Redesign

Giving an outdated website a new look and feel while also making it easier to navigate.

Affinity Diagram

Key Findings:

Purpose of the organization and website is not immediately clear

Users described the website as “childish”

Users don’t trust the website to volunteer with or make donations, leading to distrust with the organization

Persona

We decided Sam, a teacher was our user planning an upcoming field trip for her students.

We built the Empathy map around issues she is experiencing when navigating the current website.

Site Map

User testing identified issues with navigation confusion; simplifying and reorganizing pages by hierarchy was key.

Style Guide

We refined a style guide to help maintain consistency across the site.

Wireframes

We began with sketching our wireframes. Those were then turned into lo-fi wireframes.

User Testing

We conducted a several user tests. User were asked to locate volunteer information, identify opportunities for field trips and locating the contact page.

Hi-Fi Prototype

Conclusion
+ Next Steps

The current Tucson Village Farm website is outdated and is in much need of a refresh

This solution addresses multiple different users from admins, existing users, and new users of the website and communicates a youthful friendliness without being “childish”

This redesign also allows users to quickly identify the purpose of the organization and locate requested information

There is still a lot of potential for other pages to be redesigned for simplicity and for content to be combined
